# Haritalama

Basit rota hesaplama ve görselleştirme projesi (Flask + OSMnx + Folium).

Çalıştırma:
1. Sanal ortam oluşturun:
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1

2. Gereksinimleri yükleyin:
   pip install -r requirements.txt

3. Uygulamayı başlatın:
   python haritalamaGERL.py

Not: OSMnx ağır veri çekebilir; ilk çalışmada biraz zaman alır.
